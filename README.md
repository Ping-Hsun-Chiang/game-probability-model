# Game Probability Model

數學驅動的遊戲機率模型實驗集。每個子項目從 **數學建模 → 機率推導 → 蒙地卡羅模擬**，完整呈現遊戲設計背後的統計邏輯。

---

## 📂 模型列表

### 🎰 Slot Game — RTP 95% 數學模型
> 設計一套 RTP = 95%、高波動性的老虎機數學模型。從賠付表設計、Variance 推導、Hit Frequency 計算，到以 10,000 位玩家 × 最多 2,000 手的 Monte Carlo 模擬驗證，完整呈現遊戲數學的設計與風險控管邏輯。

- **核心參數**：RTP 95% ｜ Volatility σ ≈ 7.77 ｜ Hit Frequency 28.3% ｜ Jackpot 1,000×
- **模擬規模**：10,000 位玩家，約 1,010 萬次 Spin
- 📄 程式碼：[RTP_95.ipynb](./RTP_95.ipynb)
- 🌐 文章介紹：[Slot Game Probability Model](https://ping-hsun-chiang.github.io/personal-website/posts/game_probability_model/)
